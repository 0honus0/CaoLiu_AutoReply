# CaoLiu_AutoReply

### 用前须知：合理使用，得码不易，且行且珍惜。

### 使用说明

- 安装依赖：

  ```shell
  pip install -r requirements.txt -t ./
  ```

- 将配置文件`config.example.yml`，重命名为`config.yml`，按照提示修改内容即可

- 运行：

  ```shell
  python AutoReply.py
  ```
